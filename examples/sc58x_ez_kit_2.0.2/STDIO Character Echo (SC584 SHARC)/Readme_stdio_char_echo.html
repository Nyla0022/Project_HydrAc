1. Description

This example is a simple illustration of the use of the STDIO service to provide character echo over the UART controller.

2. Processor

  • ADSP-SC573, ADSP-SC584, or ADSP-SC589

3. Tested with

  • ADSP-SC573 EZ-Kit
  • ADSP-SC584 EZ-Kit
  • ADSP-SC589 EZ-Kit
  • Analog Devices CrossCore Embedded Studio 2.3.0

4. Overview

The example uses the STDIO service to re-direct all STDIO streams to the UART. Thereafter, the example program echoes all characters typed on to the console window connected to the UART device.

5. Hardware Setup

  • ADSP-SC573 or ADSP-SC584 EZ-Kit
    • Connect Micro USB cable from P7 (USB to UART) on the EZ-Kit to your PC

  • ADSP-SC589 EZ-Kit
    • Connect Micro USB cable from P8 (USB to UART) on the EZ-Kit to your PC

Open PuTTY/TeraTerm and configure it with the settings in Table 1.

The appropriate COM port number can be found in the Windows Device Manager under the Ports (COM & LPT) node.

Table 1: COM Settings
ConfigurationValue
Bits per second57600
Data bits8
ParityNone
Stop bits1
Flow ControlNone

6. Software switch settings

The following UART-related switches have been set:

  • ~UART0_EN set to 0
  • ~UART0_FLOW_EN set to 0

See SoftConfig_SC573.c, SoftConfig_SC584.c, or SoftConfig_SC589.c for more information.

Please refer to the EZ-KIT Lite Evaluation System Manual for information on software-controlled switches.
